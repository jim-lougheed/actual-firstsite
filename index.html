My Projects

KitchenCompanion

KitchenCompanion Customizable Cookbook is a React web application that assists users in finding countless recipe ideas based on ingredients on hand and helps them decide "What am I going to make for dinner tonight?"

The Simon Game

A challenge project from Angela Yu's Complete Web Development Bootcamp, this simple JavaScript game is simple to play. With each turn, a square will light up and add to a sequence. It is your job to remember the sequence and click the corresponding squares as it continues to get longer and longer.

Dino Game

This clone of the Chrome Dino game is based on a JavaScript tutorial from Web Dev Simplified but created using React. Press any key to start running and jump over the oncoming cacti!
